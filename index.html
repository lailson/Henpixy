Sobre o Projeto

Henpixy é uma aplicação de processamento digital de imagens desenvolvida em Python com interface gráfica baseada em PySide6 (Qt6). Projetada para fins educacionais e práticos, a aplicação permite aplicar diversas técnicas de processamento de imagens, visualizar resultados em tempo real e exportar imagens processadas.

Com uma tela de boas-vindas intuitiva, opções para imagens de amostra e uma interface limpa e moderna, Henpixy torna o processamento de imagens acessível tanto para estudantes quanto para profissionais.

Funcionalidades

Formatos de Imagem

Suporte para diversos formatos: PNG, JPEG, BMP, GIF, TIFF, WebP, ICO, PSD, XBM, XPM

Interface Intuitiva

Interface moderna com tela de boas-vindas, menus organizados e atalhos de teclado

Zoom Avançado

Ferramentas de zoom para inspeção detalhada da imagem com controles intuitivos

Informações da Imagem

Visualização de detalhes técnicos como dimensões, tipo, espaço de cores e metadados EXIF

Histogramas

Visualização de histogramas com estatísticas detalhadas e função de distribuição acumulada

Equalização

Equalização de histograma com visualização comparativa entre imagem original e equalizada

Transformações Básicas

Negativo, transformação gama, alargamento de contraste e filtro de intensidade zero

Planos de Bits

Fatiamento por planos de bits com visualização individual de cada plano e detecção automática de profundidade

Pseudocores

Fatiamento por intensidades para pseudocores com mapas predefinidos e personalização

Filtros Espaciais

Filtros da média, ordem estatística (mediana, mínimo, máximo) e Laplaciano

Aguçamento de Imagens

Aguçamento de imagens utilizando o operador Laplaciano com controle de intensidade

Inspeção de Pixels

Visualização da intensidade de pixels individualmente selecionados com matriz interativa

Histórico

Sistema de histórico para acompanhamento e reversão de modificações

Imagens de Amostra

Acesso rápido a imagens de amostra para testar as funcionalidades do aplicativo

Instruções para macOS

Se ao tentar abrir o aplicativo aparecer a mensagem "Henpixy.app está danificado e não pode ser aberto", siga um dos métodos abaixo:

Método 1 (Recomendado):

  1. Clique com o botão direito (ou Control+clique) no aplicativo Henpixy.app
  2. Selecione "Abrir" no menu de contexto
  3. Na janela de aviso, clique em "Abrir" novamente
  4. O aplicativo será executado normalmente e nas próximas vezes poderá ser aberto com duplo clique

Método 2 (Via Terminal):

  1. Abra o Terminal (em Aplicativos > Utilitários)
  2. Digite o comando abaixo (substitua pelo caminho correto se necessário): 
    xattr -d com.apple.quarantine /Applications/Henpixy.app
  3. Agora o aplicativo deve abrir normalmente

Nota: Essas restrições de segurança são impostas pelo macOS para proteger contra software não assinado. O Henpixy é um aplicativo seguro, desenvolvido para fins educacionais.

  • ✅ Fatiamento por planos de bits
  • ✅ Equalização de histograma
  • ✅ Pseudocores por fatiamento de intensidades
  • ✅ Filtro de suavização da média
  • ✅ Filtros estatísticos (média, mediana, mínimo, máximo)

Ferramentas de Visualização

  • ✅ Histogramas interativos

  • ✅ Comparação antes/depois para transformações
