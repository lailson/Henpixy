Sobre o Projeto

Henpixy é uma aplicação de processamento digital de imagens desenvolvida em Python com interface gráfica baseada em PySide6 (Qt6). Projetada para fins educacionais e práticos, a aplicação permite aplicar diversas técnicas de processamento de imagens, visualizar resultados em tempo real e exportar imagens processadas.

Python PySide6 Processamento de Imagens Pillow NumPy Matplotlib

Funcionalidades

Formatos de Imagem

Suporte para diversos formatos: PNG, JPEG, BMP, GIF, TIFF, WebP, ICO, PSD, XBM, XPM

Interface Intuitiva

Interface simples com menus e atalhos para fácil navegação e aplicação de ferramentas

Zoom Avançado

Ferramentas de zoom para inspeção detalhada da imagem com controles intuitivos

Informações da Imagem

Visualização de detalhes técnicos como dimensões, tipo, espaço de cores e metadados EXIF

Histogramas

Visualização de histogramas com estatísticas detalhadas e função de distribuição acumulada

Equalização

Equalização de histograma com visualização comparativa entre imagem original e equalizada

Transformações

Negativo, transformação gama, alargamento de contraste e outras técnicas de processamento

Planos de Bits

Fatiamento por planos de bits com visualização individual de cada plano

Inspeção de Pixels

Visualização da intensidade de pixels individualmente selecionados

Histórico

Sistema de histórico para acompanhamento e reversão de modificações

Instalação e Uso

Requisitos

  • Python 3.12 ou superior
  • PySide6 (Qt6)
  • Pillow
  • NumPy
  • Matplotlib

Instalação para Desenvolvimento

Clone o repositório e instale as dependências:

git clone https://github.com/seu-usuario/henpixy.git
cd henpixy

# Usando conda para criar um ambiente virtual
conda create --prefix ./env python=3.12
conda activate ./env

# Instalando o projeto em modo desenvolvimento
pip install -e .

Executando o Aplicativo

python -m henpixy.main

Download

Você pode baixar versões pré-compiladas para Windows e macOS na seção Releases do GitHub.

Módulos e Funcionalidades

Operações Implementadas

  • ✅ Intensidade zero: atribuição de intensidade zero a todos os pixels
  • ✅ Função negativo: implementação da transformação S = L - 1 - r
  • ✅ Transformação de potência: ajuste de gama com parâmetros personalizáveis
  • ✅ Alargamento de contraste: transformação linear por partes com pontos (r1,s1) e (r2,s2)
  • ✅ Fatiamento por planos de bits: visualização individual de cada plano
  • ✅ Equalização de histograma: com visualização comparativa

Ferramentas de Visualização

  • Histogramas interativos
  • Estatísticas detalhadas da imagem
  • Visualização de intensidade de pixels
  • Comparação antes/depois para cada transformação
